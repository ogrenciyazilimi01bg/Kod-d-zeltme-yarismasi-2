# Deneme 
önce terminale bu kodu yazın
Geçici olarak script çalıştırma izni ver

##  Yönetici PowerShell’de şu komutu çalıştır:
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
#### Bu sadece o PowerShell oturumu için geçerli, bilgisayarı riske atmaz.
#### Terminale Aynı PowerShell oturumunda aşağıdaki kodu yapıştırın:
venv\Scripts\activate
#### Son olarak bu kodu yazın
python app.py
### Şimdi tarayıcıya bu yazıyı açın
http://127.0.0.1:5000/
